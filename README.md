# Print Module Assignment

## Features
- Upload PDF
- Auto page count
- Print options (B/W, Color)
- Copies input
- Price calculation
- Save orders
- Orders list

## Tech Stack
- React (Vite)
- Spring Boot
- MySQL

## Run Project

### Frontend
cd frontend
npm install
npm run dev

### Backend
cd backend
mvn clean install
mvn spring-boot:run
